#   基础
##  Python有五个标准的数据类型：
1.  Numbers（数字）
    *   int（有符号整型）
    *   long（长整型[也可以代表八进制和十六进制]）
    *   float（浮点型）
    *   complex（复数）
2.  String（字符串）
    *   从左到右索引默认0开始的，最大范围是字符串长度少1
    *   从右到左索引默认-1开始的，最大范围是字符串开头
3.  List（列表）
4.  Tuple（元组）
5.  Dictionary（字典）

##  判断
1.  指定任何非0和非空（null）值为true，0 或者 null为false
